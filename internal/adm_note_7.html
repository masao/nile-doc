プリンタ2kou8100を登録した

非PostScriptプリンタであるEPSON LP8100Rを登録した。 みんなでプリンタをつなごう（Solaris2編）を参考にして、 以下の手順で行った。

  1. terminfo にエントリを登録する

	    	-D "2階共同研究室" \
	    	-v /dev/null -T epson-lp8100 -I epson-lp8100
	    % sudo enable 2kou8100
	    % sudo accept 2kou8100
  3. フィルタの入手

    GhostScriptでプリンタ特有の形式（ESC/Page）に変換するフィルタをかませる。これにはいくつかやり方があるようだが、 エプソンコーワから提供されているGSドライバを使う。

    他に、 EPSON ESC/Page 用ghostscriptドライバー（gdevepag）や LIPS & ESC/Page & NPDL & RPDL 対応 Ghostscript デバイスドライバなどもある。 

    	    % cd /usr/local/libexec
	    % sudo vi gsfilter.sh
	    % cat gsfilter.sh
#!/bin/sh
# output filter for EPSON LP-series:

GS="/usr/local/bin/gs"

SIZE=a4
RES=600
DUPLEX="-dDuplex"	# デフォルトは両面印刷

ARG="-q -dNOPAUSE -dSAFER -dBATCH"
ARG="$ARG -sPAPERSIZE=$SIZE -sOUTPUTFILE=- -r$RES"

# オプション解析
set -- `getopt T:sf $*`
if [ $? != 0 ]; then
  exit 2
fi

for i in $*; do
  case $i in
    -T)
      case $2 in
	*lp8100) DEV=lp8100;;
	*lp9400) DEV=lp9400;;
      esac;
      shift 2;;
    -f) DUPLEX="-dDuplex -DTumble";;
    -s) DUPLEX= ;;
  esac
done

ARG="$ARG $DUPLEX -sDEVICE=$DEV -"

$GS $ARG
	    % sudo chmod 755 gsfilter.sh

ディスプレイ交換
seine ハードディスクを認識しなくなり、活動停止。
そこで、seine のグラフィックカードをnile に移植し、seine のディスプレイを nile で使うこととする。借りていたJ3100 は返却。

手順:

  1. nile 電源停止。
  2. 裏側向かって右から2番目のスロットを抜く。
  3. seine から抜いたグラフィックカードを指定の場所にさす。
  4. 本体にスロットを戻す。
  5. 再起動 (boot -r)
    -r は新規にデバイスを接続した時に使用。
    システムのコンフィギュレーションを自動的に行ってくれる。
    デバイスの増設・取り外し時に1度行えばよい。
    → see solaris2.5.1 システム管理入門 p.51
