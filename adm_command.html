コマンド一覧。
伊東のメモ用です。
  • uname -a
    現在のシステム名の表示。 -a システムから現在、入手できる基本情報を出力。
  • hostname
    現在のホストシステムの名前を表示。
  • domainname
    現在のドメインネームの表示。 ulis や nuch が出力される。
  • more /etc/netmasks
    netmask が書いてある。
  • fsadmin
    %fsadmin -e #init.d により自動的に起動。 -d は自動的に起動しないように設定。 %fsadmin だけの時は、現在の状態を表示。
  • netstat -a
    全てのソケットとルーティングエントリーの状態を表示。
  • nslookup
    IPadress とホスト名がわかる。
  • patchadd
    patchadd patch名 でpatch が追加される。 patchadd -p 現在適用されているパッチをリスト表示する。
  • patchrm
    patchrm patch名 で システムからパッチを削除して、パッチ適用前のファイルを復元するらしい。
  • shutdown
    /usr/sbin/shutdown -y -i 0 を多く使ってます。
    詳しくはman を。
  • バックアップとリストア
    DDS3 のテープは　圧縮なしで12G、圧縮ありで24G のデータが入ります。
    DDS4 のテープは　圧縮なしで 20G、圧縮ありで 40G のデータが入ります。
    • ufsdump
      • 圧縮なしの例
        /usr/sbin/ufsdump 0ucf /dev/rmt/0mn /home
      • 圧縮ありの例
        /usr/sbin/ufsdump 0ucf /dev/rmt/0cn /home
      • ufsdump のオプション
        • 0：バックアップレベル0。完全バックアップ。
        • u：ダンプレコード(/etc/dumpdates) のアップデート。
        • c：カートリッジテープにバックアップ作成。
        • f：
        • dev/rmt/0：テープデバイス名
        • m：中密度
        • c：圧縮
        • n：バックアップ完了後、巻き戻ししない
      • ufsdump S filesystem filesysytem のバックアップのサイズをバイト単位で見積もる。
    • ufsrestore
      • テープの中身を確認したい時
        #ufsrestore tvf /dev/rmt/0 | more
      • 対話モード
        #ufsrestore i
      • １つのテープで / → /usr → /var の順にバックアップを取っており /var をリストアしたい時。 オプション s n n番目のバックアップファイルまでスキップする。 #ufsrestore ifs /dev/rmt/0 3
      • mt
        テープデバイスを操作するために、mtというコマンドが用意されている。
        • テープの巻き戻し
          mt -f /dev/rmt/0 rewind
        • 1台目のテープデバイスで、1ファイル先に進む。
          mt -f /dev/rmt/0n fsf 1
        • テープドライブの状態を表示する。
          mt -f /dev/rmt/0n status
