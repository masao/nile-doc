  1. MATLAB の使い方


     MATLAB は　グラフィックス、画像処理ができる総合的な数値解析ソフトウェアです。４ライセンスあるので、４人まで同時に使用できます。
    現在 ver.6.1 がインストールされています。(2001/08/22 現在）
    • 利用について
        MATLAB を利用するためには以下のコマンドを実行してください。
      • nile 以外の端末から利用する場合
        host% echo $DISPLAY
        your-display:0.0
        host% xhost +
        host% telnet nile.ulis.ac.jp
        nile% setenv DISPLAY your-display:0.0
        nile% matlab
      • nileの端末から利用する場合
        nile% matlab

    • マニュアルについて
        MATLAB にはオンラインのドキュメントがついています。
        ヘルプメニューの" MATLAB help "選択するか、またはMATLABを起動し
        　　　　　>> helpdesk
        で、 HTML 形式のマニュアル(英語版)が読めます。

        HTML 形式の日本語版help があります。しかし、MATLABのソフト上でファイルを開いても
        文字化けしてしまうので、日本語版helpを読みたい場合は、
        nile の Netscape を起動し、 以下のファイルを指定してください。
        　　　　　file:/usr/local/matlab/jhelp/helpdesk.html
        また、MATLAB の冊子体のマニュアルは２階の共同研究室に置いてあります。

    • MATLAB5を利用するためには
        MATLAB5(5.3.1)が nile:/usr/local/matlab5 にあります。
        利用する時には /usr/local/matlab5/bin/matlab を指定するか、
        または、~/.cshrc の path 行で /usr/local/bin より前のところに
        /usr/local/matlab5/bin を書き加えてください。

  2. MALTAB Webserver


    Matlab Webserver を利用することで、Matlab のプログラムを Webserverを介して実行したり、結果のグラフをブラウザ上に 表示することができます（→ デモがあります）。
    プログラム、htmlファイルの書き方などは Mathworksのホームページにあるマニュアルを見てください。

    • jpegファイル作成ディレクトリのパーミッションの変更

      Matlab webserverを利用してjpegファイルを作成する場合は、 そのファイルが作成されるディレクトリのパーミッションを777にする 必要があります。

    • Matlab Webserver が提供しているデモプログラム (webpeaks, webmagic, Softball Statistics)を
      個人のディレクトリで実行するためには。

      Matlab Webサーバのポート番号: 27009 、*.html, *.m, 実行ファイルの置き場所、jpegファイルの作成場所を
      /home/chinatsu/public_html/matlab とします。

      1. /usr/local/matlab6p1/toolbox/webserver/wsdemos にあるファイルを
        /home/chinatsu/public_html/matlab にコピーします。
        % pwd
        /home/chinatsu/public_html/matlab
        % cp /usr/local/matlab6p1/toolbox/webserver/wsdemos/* .

      2. matweb がCGIプログラムになっています（バイナリなので中身はみれません）。
        nileの設定では cgiのPOSTメソッドを利用する場合、 呼び出されて実行されるプログラム名が *.cgi でなければ
        いけないので matweb の名前を matweb.cgi に変更します。

        % mv matweb matweb.cgi

        MathWorks のロゴを表示させたい場合は ./icons ディレクトリを作成し、MathWorks.Logo.gif, mathby.gif,
        peaks-jr.jpg を./icon にコピーします。次に、*.htmlの src="/icons/ロゴファイル名" を src="./icons/ロゴファイル名" に
        直してください。

      3. matweb.conf.org を編集します。

        mlserver=<matlabserver_host_name>,
        を
        mlserver = nile　　に、
        mldir=<matlab>/toolbox/webserver/wsdemos
        を
        mldir = /home/chinatsu/public_html/matlab
        に書き換えます。
        各アプリケーションのフィールドに
        mlport=27009
        を追加します。

      4. .html, .m ファイルを編集します。
        Peaks plotの変更点です。

        % diff peaksplot.html.org peaksplot.html
        5c5
        < <form action="/cgi-bin/matweb" method="POST" target="outputwin">
        ---
        > <form action="./matweb.cgi" method="POST" target="outputwindow">

        % diff webpeaks.m.org webpeaks.m
        55c55
        < s.GraphFileName = sprintf('%speaks.jpeg', mlid);
        ---
        > s.GraphFileName = sprintf('./%speaks.jpeg', mlid);
        57c57
        < s.GraphFileName = sprintf('/icons/%speaks.jpeg', mlid);
        ---
        > s.GraphFileName = sprintf('./%speaks.jpeg', mlid);

        MATLAB Matrix display は webmagic1.htmlを修正してください。
        % diff webmagic1_org.html webmagic1.html
        30c30
        < <form action="/cgi-bin/matweb" method="POST">
        ---
        > <form action="./matweb" method="POST">

        Softball Statistics は index.html を修正してください。。
        % diff index_org.html index.html
        51c51
        < <li><a href="/cgi-bin/matweb?mlmfile=players">Softball
        ---
        > <li><a href="./matweb?mlmfile=players">Softball

      5. webpeaksでjpegファイルが作成されるので、ディレクトリのパーミッションを777に変更します。
        %chmod 777 /home/chinatsu/public_html/matlab

  3. SUN 提供のコンパイラ

    nile では Sun WorkShop Compilers 4.2 で提供されているFORTRAN(f77,f90), C++,C のコンパイラが利用できます。
    • C でワイド文字を処理するには。
      Sun WorkShop Compilers 4.2 の Cコンパイラでワイド文字を処理するためには、
      オプションで -lw をつけてください。

  7. f2c

    f2c は Fortran77 から C,C++へのコンバータです。

    実行例）

    % f2c hello.f
    hello.f:
    MAIN:
    % cc hello.c -lf2c -lm
    % a.out
    Hello World!

    f2c を実行すると *.c が作成されます。
    cc の時は、オプションとして -lf2c -lm が必要です。 その他 f2c については man を見てください。

    f2c のテクニカルレポート(1995年のもの）が nile:/usr/local/doc/f2c.ps にあります。

  8. JLAPACK

    JLAPACK はLapack をJavaに移植したものです。 nile:/usr/local/jlapack-0.3a にインストールしてあります。 jlapack-0.3aを利用するためには、ログイン毎に

    % setenv CLASSPATH .:/usr/local/jlapack-0.3a/classes.zip

    を実行するか、または、 ~/.cshrc などのシェルの設定ファイルに

    setenv CLASSPATH .:/usr/local/jlapack-0.3a/classes.zip

    の一行を加え、

    % soruce 設定ファイル名

    を実行してください。
