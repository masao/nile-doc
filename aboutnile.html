nile について

nile の本体は研究開発室1 においてあります。 そのため、dream 用のX 端末、第２講座の共用SUN ワークステーションなどから アクセスすることになります。

nile の本体は Sun Ultra-Enterprise3000 というマシンです。
  • OS: Solaris7 ( SunOS 5.7 )
  • CPU : 250MHz UltraSPARC 2つ
  • メモリ : 512MB（256MB *2）

nile のアカウントについて

nile のアカウントは本学の学生・教官およびその関係者より申し出があれば、
ID を登録しています。 nile の利用を希望する方は admin@nile.ulis.ac.jp までご連絡下さい。
